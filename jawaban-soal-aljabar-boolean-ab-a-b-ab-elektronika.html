10. (A+BC)(A+C)+(A+B)'(A+C) cara menjawab dari aljabar boole ini


(A + BC) (A + C) + (A + B) (A + C)

(A² + C + BC) + (A² + B + C)

PEMBAHASAN

Kita kalikan terlebih dahulu variabel yang sama yaitu : A × A = A²

Karena BC dan C Variabelnya berbeda maka tidak usah dikalikan lagi, jadi tinggal ditambahkan saja yaitu : A² + C + BC

Sama seperti sebelumnya Kita kalikan terlebih dahulu variabel yang sama yaitu : A × A = A²

Karena B dan C variabelnya berbeda maka tidak usah dikalikan lagi, jadi tinggal ditambahkan saja yaitu : A² + B + C

KESIMPULAN

Jadi, aljabar dari (A + BC) (A + C) + (A + B) (A + C) adalah (A² + C + BC) + (A² + B + C)

18. Buatlah gambar rangkaian logika Boolean pada persamaan output berikut : y = (AB + AC). ABC​


Rangkaian logika ini terdiri dari tiga gerbang AND dan satu gerbang OR. Gerbang AND pertama menerima input dari A dan B, dan gerbang AND kedua menerima input dari A dan C. Gerbang OR menerima input dari kedua gerbang AND, dan outputnya adalah y.

Penjelasan:

Gerbang AND pertama menghasilkan output 1 jika A dan B keduanya 1.

Gerbang AND kedua menghasilkan output 1 jika A dan C keduanya 1.

Gerbang OR menghasilkan output 1 jika salah satu atau kedua inputnya adalah 1.

Dengan demikian, output y akan menjadi 1 jika A, B, dan C semuanya 1, atau jika salah satu dari A, B, atau C adalah 1 dan yang lainnya adalah 0.

Contoh:

Jika A = 1, B = 0, dan C = 1, maka output y adalah 0.

Jika A = 1, B = 1, dan C = 1, maka output y adalah 1.

Jika A = 0, B = 1, dan C = 1, maka output y adalah 1.

Untuk membuktikan kesamaan (a+b)(a+c) = a+bc tanpa menggunakan tabel kebenaran, kita dapat menggunakan hukum De Morgan dan prinsip dualitas. Berikut adalah langkah-langkahnya:

1. Buktikan menggunakan Hukum De Morgan:

  Dengan Hukum De Morgan, kita dapat mengubah operasi perkalian menjadi operasi penjumlahan dan sebaliknya. Berikut adalah langkah-langkahnya:

  (a+b)(a+c) = a + ab + ac + bc  (distributif)

             = a(1+b+c) + bc    (distributif)

             = a(1) + bc         (a+a = a dan a+1 = 1)

             = a + bc

  Dengan menggunakan Hukum De Morgan, kita dapat membuktikan bahwa (a+b)(a+c) = a+bc.

2. Buktikan menggunakan Prinsip Dualitas:

  Prinsip Dualitas menyatakan bahwa jika suatu persamaan boolean benar, maka persamaan yang diperoleh dengan menukar operasi penjumlahan dan perkalian serta mengganti 0 dengan 1 dan sebaliknya juga benar.

  Kita dapat menggunakan prinsip dualitas untuk membuktikan kesamaan ini. Berikut adalah langkah-langkahnya:

  Diberikan: (a+b)(a+c) = a+bc

  Dualitas:

  - Ganti operasi penjumlahan dengan perkalian dan sebaliknya.

  - Ganti 0 dengan 1 dan sebaliknya.

  Kita akan mengganti penjumlahan dengan perkalian dan sebaliknya, serta mengganti 0 dengan 1 dan sebaliknya pada persamaan tersebut:

  (a*b)+(a*c) = a+b*c

  Dalam persamaan ini, kita dapat melihat bahwa kedua sisi persamaan adalah sama setelah mengikuti prinsip dualitas. Oleh karena itu, kesamaan (a+b)(a+c) = a+bc benar.

Dengan demikian, kita telah membuktikan kesamaan (a+b)(a+c) = a+bc menggunakan hukum De Morgan dan prinsip dualitas tanpa menggunakan tabel kebenaran.

38. Fungsi Aljabar Boolean: a. F(A,B,C) = A(BC' + C) + B(A + A'C)​


Jawab:

Fungsi Aljabar Boolean merupakan fungsi yang menggunakan simbol-simbol dari aljabar Boolean untuk menentukan apakah suatu pernyataan benar atau tidak. Fungsi tersebut dapat dituliskan dengan menggunakan variabel-variabel yang mewakili suatu keadaan atau nilai, serta operator-operator logika seperti AND, OR, dan NOT.

Fungsi Aljabar Boolean yang diberikan adalah:

F(A,B,C) = A(BC' + C) + B(A + A'C)

Untuk memahami makna dari fungsi tersebut, pertama-tama Anda perlu memahami makna dari setiap simbol yang digunakan. Di sini, A, B, dan C adalah variabel yang mewakili suatu keadaan atau nilai. ' merupakan operator NOT, yang berarti negasi atau kebalikan dari suatu nilai. Operator AND dituliskan dengan tanda kurung, sedangkan operator OR dituliskan dengan tanda +.

Jadi, fungsi tersebut dapat diterjemahkan sebagai berikut:

F adalah hasil dari A AND (B AND NOT C OR C) OR B AND (A OR NOT C)

Fungsi tersebut menentukan apakah suatu pernyataan benar atau tidak berdasarkan keadaan atau nilai dari variabel A, B, dan C. Nilai F akan bernilai 1 (benar) jika salah satu dari pernyataan di dalam kurung kurawal bernilai 1 (benar), atau jika salah satu dari pernyataan di dalam kurung kurawal bernilai 0 (salah) dan pernyataan di dalam kurung kurawal kedua bernilai 1 (benar). Sebaliknya, nilai F akan bernilai 0 (salah) jika kedua pernyataan di dalam kurung kurawal bernilai 0 (salah).
