14. Kunci jawaban kelas xl smk matematikanilai dari (2n+1) bila n =11 adalah​


Jawaban:

23

Penjelasan dengan langkah-langkah:

Substitusikan nilai n ke (2n + 1)

= 2(11) + 1

= 22 + 1

= 23

Maka, nilai dari (2n + 1) bila n = 11 adalah 23

Detail Jawaban:Mapel : MatematikaKelas : 8 SMPMateri : Bab 3 - Sistem PersamaanKata kunci : nilai dari 2n + 1Kode soal : 2kode kategorisasi : 8.2.3

Nilai dari (2n+1) bila n = 11 adalah 23.

Pendahuluan :

Persamaan atau Pertidaksamaan Satu Variabel merupakan persamaan atau pertidaksamaan dalam matematika yang hanya memuat satu variabel dengan pangkat terkecil yaitu satu.

Persamaan Linear Satu Variabel adalah suatu persamaan yang menggunakan tanda sama dengan (=). Sedangkan Pertidaksamaan Linear Satu Variabel menggunakan tanda (< , ≤ , > , ≥). Pada saat mengalikan salah satu ruas dengan bilangan negatif, maka angka pertidaksamaan dibalik. Contoh : ("<" menjadi ">") ataupun ("≥" menjadi "≤")

Pembahasan :

Diketahui :

n = 11

Ditanya :

Nilai dari (2n+1)

Jawab :

Subtitusi nilai n = 11 :

[tex] \rm = 2n+1[/tex]

[tex] \rm = 2(11)+1[/tex]

[tex] \rm = 22+1[/tex]

[tex] \bf = 23[/tex]

Kesimpulan :

Jadi, hasilnya adalah 23.

40. identitas trigonometri kelas xl ​


Pembahasan

Identitas Trigonometri

[tex] \boxed{ \sin(2 \alpha ) = 2 \sin( \alpha ) \cos( \alpha ) } \\ \boxed{ \cos(2 \alpha ) = 1 - 2 { \sin }^{2} ( \alpha ) } \\ \boxed{ \tan(2 \alpha ) = \frac{ \sin(2 \alpha ) }{ \cos(2 \alpha ) } } \\ \\ [/tex]

Diketahui :

[tex] \sin( \alpha ) = \frac{24}{25} \: \: \: , \: \: 0 < \alpha < \frac{\pi}{2} \\ \\ [/tex]

Ditanya :

[tex]a. \sin(2 \alpha ) \\ \\ b. \cos(2 \alpha ) \\ \\ c. \tan(2 \alpha ) \\ \\ [/tex]

Penyelesaian :

[tex] \sin( \alpha ) = \frac{24}{25} \\ \\ \cos( \alpha ) = \frac{ \sqrt{ {25}^{2} - {24}^{2} } }{25} = \frac{ \sqrt{49} }{25} = \frac{7}{25} \\ \\ [/tex]

[tex]a. \\ \sin(2 \alpha ) = 2 \sin( \alpha ) \cos( \alpha ) \\ \\ \sin(2 \alpha ) = 2 \cdot \frac{24}{25} \cdot \frac{7}{25} = \frac{336}{625} \\ \\ [/tex]

[tex]b. \\ \cos(2 \alpha ) = 1 - 2 { \sin }^{2} ( \alpha ) \\ \\ \cos(2 \alpha ) = 1 - 2 { \left ( \frac{24}{25} \right )}^{2} \\ \\ \cos(2 \alpha ) = 1 - 2 \left ( \frac{576}{625} \right ) \\ \\ \cos(2 \alpha ) = 1 - \left ( \frac{1152}{625} \right ) \\ \\ \cos(2 \alpha ) = - \frac{527}{625} \\ \\[/tex]

[tex]c. \\ \tan(2 \alpha ) = \frac{ \sin(2 \alpha ) }{ \cos(2 \alpha ) } \\ \\ \tan(2 \alpha ) = \frac{ \frac{336}{625} }{ - \frac{527}{625} } \\ \\ \tan(2 \alpha ) = - \frac{336}{527} \\ \\ [/tex]
