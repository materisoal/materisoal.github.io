- Baju : 80.000 × 25/100 = 20.000. Jadi, 80.000 - 20.000 = 60.000
- Celana : 100.000 × 10/100 = 10.000. Jadi, 100.000 - 10.000 = 90.000
Jadi, harga total 60.000 + 90.000 = 150.000.

Untuk Toko Damai :
- Baju : 80.000 × 20/100 = 16.000. Jadi, 80.000 - 16.000 = 64.000
- Celana : 100.000 × 15/100 = 15.000. Jadi, 100.000 - 15.000 = 85.000.
Jadi, harga total = 64.000 + 85.000 = 149.000.

Untuk Toko Senang :
- Baju : 80.000 × 15/100 = 12.000. Jadi, 80.000 - 12.000 = 68.000.
- Celana : 100.000 × 20/100 = 20.000 .Jadi, 100.000 - 20.000 = 80.000.
Jadi, harga total = 68.000 + 80.000 = 148.000.
Untuk Toko Indah :
- Baju : 80.000 × 10/100 = 8.000. Jadi, 80.000 - 8.000 = 72.000.
- Celana : 100.000 × 25/100 = 25.000. Jadi, 100.000 - 25.000 = 75.000
Jadi, harga total = 72.000 + 75.000 = 147.000.

Kita dapat menyimpulakn bahwa toko yang menawarkan harga paling murah adalah = D. Tokoh Indah dengan harga total Rp. 147.000.

kenaikan peserta laki : 5 %
kenaikan peserta prempuan 20%
kenaikan keseluruhan peserta 10%

misal :
a = jumlah peserta laki-laki th lalu = 300.000
b = jumlah peserta perempuan th lalu
P = jumlah peserta keseluruhan

P = a + b

persamaan kenaikan peserta dapat dijabarkan sbb :
peserta laki2 x 5% + peserta perempuan x 10% = peserta keseluruhan x 10%

5a + 20b = 10P
5a + 20b = 10(a + b)
5a + 20b = 10a + 10b
20b - 10b = 10a - 5a
10b = 5a
2b = a

dapat dikatakan bahwa jumlah peserta laki-laki jumlahnya 2 kali peserta perempuan (tahun lalu)
maka:
jumlah peserta laki2 = 300.000
jumlah peserta perempuan = 150.000
jumlah peserta keseluruhan (tahun lalu) = 300.000+ 150.000 = 450.000

peserta tahun ini mengalami kenaikan 10%; maka peserta tahun ini(2016)
450.000 + (450.000 x 10%) = 450.000 + 45000 = 495000

34. berapa paket untuk ujian nasional smp dan sma 2016?? apakah betul soal tiap provinsi untuk un berbeda ??


Jawaban:

 

Paket untuk ujian nasional SMP dan SMA 2016 ada 32, karena soal tiap provinsi untuk UN berbeda.

 

Pembahasan:

 

Paket soal ujian nasional (UN) untuk setiap provinsi akan berbeda satu sama lainnya. Namun, standar nilai setiap provinsi akan sama. Jumlah jenis paket soal ada 32, karena ada 33 provinsi, dikurangi provinsi Kalimantan Utara yang baru terbentuk.

 

Badan Standar Nasional Pendidikan (BSNP) menyatakan bahwa hanya 25 persen dari total jumlah soal UN yang dibuat BSNP dan berlaku secara nasional. Sisanya dibuat oleh tim di masing-masing provinsi. Pembuatan soal akan disupervisi oleh BSNP.

 

Soal di setiap provinsi akan dibuat oleh para guru senior. Jadi, soal sudah disesuaikan dengan kondisi masing-masing daerah.  Namun, standar nilai kelulusan akan tetap sama untuk semua provinsi. Semua peserta wajib mendapatkan nilai rata-rata minimal 5,5 untuk setiap mata ujian.

 

Dengan kata lain, naik dibandingkan dengan tahun lalu. Penignkatan standar ini untuk meningkatkan kualitas pendidikan di Indonesia.

 

Mengutip Panduan Operasional Standar (POS) UN 2016, pelajar SMA kelas XII pakan mengerjakan:

 

·         50 butir soal Bahasa Indonesia selama 120 menit,

·         40 soal Matematika selama 120 menit dan

·         50 soal Bahasa Inggris selama 120 menit.

·         Siswa jurusan IPA: tes Fisika, Kimia dan Biologi masing-masing 40 soal selama 120 menit.

·         Siswa jurusan IPS: 40 soal Ekonomi serta masing-masing 50 soal Sosiologi dan Geografi selama 120 menit.

 

Kelas: XII

Mata pelajaran: Ujian Nasional

Materi: Ujian Nasional

Kata Kunci:  Ujian Nasional
