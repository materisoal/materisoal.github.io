9. Tuliskan masing masing 1 kalimat positif dan 1 kalimat negatif yg menggunakan quantifer Kelas 6 sd Bhs inggris


Jawaban:

I have some books

I don't have any cats.

Penjelasan:

Quantifier adalah kata yang digunakan untuk menyatakan jumlah. Contoh yg kita gunakan adalah some and any. Kedua nya bisa digunakan baik unt benda yg dapat dihitung(countable nouns) atau benda yg tidak dapat dihitung(uncountable nouns).

Some digunakan dalam kalimat positif.

Any digunakan dlm kalimat negatif dan kalimat tanya.

23. Ayo kak, yg pintar bhs inggris. Ini soal kel. 6 SD tolong dijawab ya kak :)


Subject : English

Category : Present Perfect Tense

Level : ES VI

Code : 5

Struktur Present Perfect :

S + have/has + past Participle (V₃)

*have untuk subjek I/you/we/they

*has untuk subjek he/she/it

Untuk pertanyaan menggunakan how long strukturnya :

How long + S + have/has + past participle

Answers :

2. How long have the Robinsons had their dog?

They've had it for five years.

3. How long has Charlie had his computer game?

He's had it for two weeks.

4. How long have the Carricks lived in Boswell Street?

They've lived there since 2002.

Soal 1:

Diberikan dua himpunan berikut:

A = {1, 2, 3, 4}

B = {3, 4, 5, 6}

Soal 2:

Diberikan relasi R = {(1, 3), (2, 4), (3, 5), (4, 6)} antara himpunan A = {1, 2, 3, 4} dan B = {3, 4, 5, 6}. Apakah relasi R merupakan relasi yang:

a) Reflexive (Refleksif)?

b) Symmetric (Simetri)?

c) Transitive (Transitif)?

Soal 3:

Diberikan relasi R = {(2, 4), (3, 6), (4, 8)} antara himpunan A = {2, 3, 4, 5, 6} dan B = {4, 6, 8, 10}. Tentukan himpunan domain dan kodomain dari relasi R.

Diberikan himpunan A = {1, 2, 3} dan B = {3, 4, 5}. Tentukanlah himpunan relasi dari A ke B yang tidak memiliki elemen.

Soal 5:

a) Reflexive (Refleksif)?
