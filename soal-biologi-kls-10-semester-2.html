Penjelasan dengan langkah-langkah:

Diketahui :

gambar di soal.

b = 2

a = √12

Ditanya:

sin B, cos B, tan B,secan B, cosecan B,cotangen B.

Pembahasan:

Pythagoras:

c² = a² + b²

Rumus:

sin B = b/c

cos B = a/c

tan B = b/a

secan B = c/a

cosecan B = c/b

cotangen B = a/b

cari dulu yang belum diketahui c.

c² = a² + b²

c² = (√12)² + 2²

c² = 12 + 4

c² = 16

c = 4

sin B = b/c

= 2/4

= ½

cos B = a/c

= √12 /4

= 2√3 /4

= ½√3

tan B = b/a

= 2/√12

= 2/(2√3)

= 1/√3 × √3 /√3

= ⅓√3

secan B = c/a

= 4/√12

= 4/(2√3)

= 2/√3 × √3 /√3

= ⅔√3

cosecan B = c/b

= 4/2

= 2

cotangen B = a/b

= √12 /2

= 2√3 /2

= √3
