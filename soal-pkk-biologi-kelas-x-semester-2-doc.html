36. 0,5937 x 10^-¹² =..Jawab yah pkk penjelasan


0,5937 x 10⁻¹² = 5,937 x 10⁻¹³

Pembahasan

Bentuk baku atau notasi ilmiah adalah untuk metode atau cara untuk menuliskan bilangan yang sangat besar atau sangat kecil. Misalkan bilangan yang mengandung banyak 0 atau bilangan yang berada di antara 0, dan 1. Bentuk baku terdiri dari dua bagian yaitu bilangan dasar a dan pangkat dari 10. Bentuk baku dituliskan dengan :

a x 10ⁿ, dengan 1 ≤ a < 10

Karena bentuk baku merupakan bilangan berpangkat maka kita juga harus mengetahui sifat dasar perpangkatan, yaitu :

[tex]\bold{a^x.a^y=a^{x+y}}\\\\\bold{a^x:a^y=a^{x-y}}[/tex]

Cara mengubah bilangan ke dalam bentuk baku.

a. Bilangan yang sangat besar

Caranya adalah tulis bilangan terdepan sebagai bilangan dasar, tambahkan koma antara bilangan dasar dengan bilangan kedua, hitung banyak bilangan dibelakang koma dan jadikan sebagai pangkat positif dari 10. Hilangkan angka nol di belakang bilangan dasar.

Contoh :

3.400 = 3,400 x 10³ = 3,4 x 10³

1.200.000 = 1,200000 x 10⁶ = 1,2 x 10⁶

2.700.000.000 = 2,700000000 x 10⁹ = 2,7 x 10⁹

b. Bilangan yang sangat kecil

Caranya adalah tulis bilangan bukan nol kemudian hitung digit angka di belakang koma dan jadikan pangkat negatif dari 10. Terakhir sesuaikan bilangan dasar agar a ≤ 1 < 10

Misal :

0,32 = 32 x 10⁻² = 3,2 x 10¹ x 10⁻² = 3,2 x 10¹⁻² = 3,2 x 10⁻¹

0,0000234 = 234 x 10⁻⁷ = 2,34 x 10² x 10⁻⁷ = 2,34 x 10²⁻⁷ = 2,34 x 10⁻⁵

====================================================

0,5937 x 10⁻¹² = 5.937 x 10⁻⁴ x 10⁻¹²

0,5937 x 10⁻¹² = 5,937 x x 10³ x 10⁻⁴ x 10⁻¹²

0,5937 x 10⁻¹² = 5,937 x 10³⁻⁴⁻¹²

0,5937 x 10⁻¹² = 5,937 x 10⁻¹³

Jadi hasil dari 0,5937 x 10⁻¹² adalah 5,937 x 10⁻¹³
